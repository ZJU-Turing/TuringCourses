# 图灵班学习指南

<div align="center">

> 折大之初，谁传道之？  
图灵数世，何由考之？  
群贤起意，壬寅奔之；  
离经习坎，八卷述之：  
思政军体，何以混之？  
通识选修，水之乐之。  
数理基础，可求活之？  
专业课程，纵狂行之。  
夏昼苦长，短学修之；  
文质农工，统略纳之。  
旷野迷途，书评引之；  
前绪绍世，一站括之。  
见此文者，愿能惜之；  
谏往得失，进退由之。  
世世得进，恒者唯知！

</div>
